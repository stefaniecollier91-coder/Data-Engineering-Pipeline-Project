# Data-Engineering-Pipeline-Project
# CVG Flight Delay and Weather Database Project

This repository contains the database deliverables for the Flight Delays and Weather Impact Analysis at Cincinnati/Northern Kentucky International Airport project.

## Project Files

- `database_schema_documentation.docx` - Documentation describing the database tables, columns, data types, primary keys, foreign keys, and table purposes.
- `er_diagram.png` - Entity Relationship Diagram showing how the database tables are connected.
- `initial_postgresql_load_script.sql` - PostgreSQL script used to create the database tables and insert initial sample data.

## Project Overview

This project analyzes the relationship between weather conditions and airline performance at CVG Airport. It combines BTS TranStats airline operational data with Open-Meteo weather data to support flight delay, cancellation, and weather impact analysis.

## Tools Used

- PostgreSQL
- Python
- Pandas
- Open-Meteo API
- BTS TranStats Data
- Power BI
